Commands list
• current
  • /current
  • Shows the music currently playing and its progress
• find
  • /find url
  • Parses the media at the given URL and returns the music used in it
• grab
  • /grab
  • Similar to the current command but sends the message in DM
• info
  • /info
  • Get information about the bot like the ping and the uptime.
• join
  • /join
  • Connect the bot to the voice channel you are in
• leave
  • /leave
  • Disconnect the bot from the current connected voice channel.
• links
  • /links
  • Send all links to support the bot
• loop
  • /loop mode
  • Allow user to enable or disable the looping of a song or queue.
• pause
  • /pause
  • Pause the current music without skipping it
• play
  • /play query top
  • Play music from URL or search
• queue
  • /queue show add remove clear shuffle
  • Base command to perform manipulations on the bot queue
• radio
  • /radio lofi jazz rocknroll chiptune synthwave lounge minecraft rap poprock workout slowed
  • Launch a random radio
• reset
  • /reset
  • Disconnects the bot and resets its internal state if something isn't working.
• resume
  • /resume
  • Resumes the current or the next music
• search
  • /search query source
  • Allows you to search the first 10 results for the desired music.
• share
  • /share url
  • Returns all streaming platforms available for the given media
• skip
  • /skip
  • Skips to the next music in the queue. Stop the music if the queue is empty
• stop
  • /stop
  • Stop and skip current music without disconnecting bot from voice channel
• watch
  • /watch
  • Launch a Youtube Together session in the voice channel where you are currently.
